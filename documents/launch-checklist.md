# Launch Checklist

## Pre-Launch

- [ ] Review all pages for accuracy and completeness
- [ ] Confirm navigation works on every page
- [ ] Validate all forms submit successfully
- [ ] Check phone, email, and external links
- [ ] Verify mobile and tablet layouts
- [ ] Review page titles and metadata

## Content

- [ ] Proofread headings, copy, and CTAs
- [ ] Check spelling, grammar, and punctuation
- [ ] Confirm brand colors and fonts are consistent
- [ ] Validate images and graphics display correctly

## Performance and SEO

- [ ] Compress and optimize images
- [ ] Minify CSS and assets where possible
- [ ] Run PageSpeed Insights for desktop and mobile
- [ ] Confirm meta descriptions and alt text are present

## Accessibility

- [ ] Ensure all images have descriptive alt text
- [ ] Confirm form fields have labels
- [ ] Check color contrast for text and buttons
- [ ] Verify keyboard navigation works

## Launch

- [ ] Deploy latest version to production
- [ ] Confirm site loads correctly after deployment
- [ ] Test main CTAs on live site
- [ ] Verify SSL certificate and secure connection

## Post-Launch

- [ ] Share launch details with client
- [ ] Monitor analytics and error logs
- [ ] Schedule follow-up support or updates
